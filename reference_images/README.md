# Reference Images

Place reference images of known medicine packages here for comparison.

## Required Images

Create placeholder images with these names:

1. `sun-pharma_azithromycin.jpg` - Sun Pharma Azithromycin 500mg
2. `cipla_ciprofloxacin.jpg` - Cipla Ciprofloxacin 500mg  
3. `dr-reddy_omeprazole.jpg` - Dr. Reddy's Omeprazole 20mg
4. `lupin_amoxicillin.jpg` - Lupin Amoxicillin 500mg
5. `aurobind_azithromycin.jpg` - Aurobindo Azithromycin 250mg

## Image Requirements

- Format: JPEG or PNG
- Size: At least 800x600 pixels
- Should clearly show:
  - Medicine name
  - Company/brand name
  - Dosage information
  - Batch number (if visible)
  - Expiry date (if visible)

## Usage

These images are used as reference data for the medicine verification system. The system compares uploaded images against these references to help identify medicines.
